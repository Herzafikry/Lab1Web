# Lab1Web
Pertama, buat file baru dengan nama **lab1_tag_dasar.html** dan tambahkan tag dasar dokumen 
HTML. Kemudian selanjutnya, buka file tersebut pada web browser misalnya **Microsoft EDGE**.

perbedaan lainnya.
